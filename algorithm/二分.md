## 二分答案

两个模板，一个是向下取整，一个是向上取整

- 向下取整 [ l , mid ] [ mid+1 ,  r]

> 因为答案一直在往左边靠(即使`mid+1`符合也舍弃)，所以可以查找大于x的第一个数

```c++
while(l<r){
    int mid=l+r >>1; (整除向下取整)
    if(check(mid)) r=mid;
    else l=mid+1;
}
cout<<l;
```

- 向上取整 [ l , mid-1] , [ mid , r ]

> 因为答案一直在往右边靠(即使`mid-1`符合也舍弃)，所以可以查找小于x的最后一个数

```c++
while (l<r){
    int mid=l+r+1 >>1;(手动 +1 ，使得向上取整)
    if(check(mid)) l=mid;
    else r=mid-1;
}
cout<<l;
```
